Grid

Este se usa para distribuir y alinear elementos dentro de un contenedor, prioncipalmente en una sola direccion (horizontal o vertical), es ideal para alinear elementos a lo largo de un eje principal y controlar su espacio de manera eficiente.

Propiedades principales:

Flex-direction: Define la direccion en la que se alinean los elementos con row, row-reverse, column, column-reverse.

Lo que hace el row es que es que alinea los elementos de forma horizontal de izquierda a derecha.

El row-reverse alinea los elementos de forma horizontal pero de derecha a izquierda.

El column alinea los elementos de forma vertical de arriba hacia abajo.
